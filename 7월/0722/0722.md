## API(application programming lnterface)
- 컴퓨터나 컴퓨터 프로그램 사이의 연결
- 일종의 소프트웨어 인터페이스이며 다른종유릐 소프트웨워에 서비스를 제공
- 사용하는 방법을 기술하는 문서나 표준은 API 사양/명세 (speecification)

- API 사용 전에는 명세서 확인 必
  - http(프로토콜) 주소로 요청을 보내면 문서로 응답 해준다
1. 요청하는 방식에 대한 이해
  - 인증방식
  - URL 생성
    - 기본주소
    - 원하는 기능에 대한 추가 경로
    - 요청 변수 (필수와 선택)
2. 응답 결과에 대한 이해
  - 응답결과 타입(JSON)
  - 응답 결과 구조
